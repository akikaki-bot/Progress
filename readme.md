# Progress

# What is progress?
progressは簡単にプログレスバーを作成することを目的としたものです。

# Install 

```
npm i https://github.com/akikaki-bot/progress
```

# How to use?
```
const { ProgressBar } = require('progress')

let pgr = new ProgressBar()

pgr.gen(10,100)
//Return [#---------]
```
